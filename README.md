# Server Pulse

Server Pulse is a lightweight server monitoring dashboard built with FastAPI, Docker, and Raspberry Pi.

## Features

- Live CPU usage
- Live RAM usage
- Temperature monitoring
- Visitor analytics
- Docker support
- Real-time dashboard

## Tech Stack

- Python
- FastAPI
- Docker
- Nginx
- SQLite
- Raspberry Pi 3B

Docker will be used for this project but you can also download all the required packages with:
pip install -r requirements.txt
